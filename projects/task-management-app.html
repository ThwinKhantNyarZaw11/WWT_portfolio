Task Management Web App

Overview

Designed and developed a full-featured task management web application to help teams organize their work, collaborate on projects, and increase productivity. Built using the MERN stack (MongoDB, Express.js, React, Node.js), the app includes user authentication, task creation/editing, real-time status updates, and role-based access control.

Key Features

  • Secure user registration and login with JWT
  • Create, edit, assign, and delete tasks
  • Real-time task updates using Socket.io
  • Search, filter, and sort tasks by deadline or status
  • Role-based access for admins and contributors
  • Responsive design for desktop and mobile

Technical Stack

  • Frontend: React, Tailwind CSS, React Router
  • Backend: Node.js, Express.js
  • Database: MongoDB with Mongoose ODM
  • Authentication: JWT, bcrypt
  • Real-time Communication: Socket.io
  • Deployment: Render for backend, Netlify for frontend

Challenges & Solutions

  • Challenge: Ensuring real-time task updates across users
  • Solution: Used Socket.io to broadcast task changes instantly across connected clients
  • Challenge: Avoiding duplicate task submissions
  • Solution: Added optimistic UI feedback and debouncing logic in React

Impact

  • Streamlined project workflows and communication for small teams
  • Used as a team tool during a university group project with real-time collaboration
  • Praised by users for its intuitive UI and quick response times
